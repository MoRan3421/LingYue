# 灵约 (Ling Yue) · 吉隆坡仙城地理志 (Lore Map)

欢迎来到“吉隆坡仙城 (KL Immortal City)”，这是数字化与古老业力交汇的奇点。

## 1. 始源中心 (Sovereign Core)
- **双子星塔 (Twin Soul Spires)**: 全服算力的终极物理承载点。
- **始源大厅**: 觉醒者首次降临的圣殿。

## 2. 竞技位面 (Combat Realms)
- **正方形野区 (The Square Wilds)**: 英雄们决战 3.5D 的荒野位面。
- **鸿蒙试炼场**: 单人挑战与 AI 训练营。

## 3. 生活与资产 (Metropolis Layers)
- **珍宝馆 (Asset Vault)**: 九道资产的实物化存储区。
- **桃园社交区**: 缔结深厚羁绊的赛博浮岛。

## 4. 无法地带 (The Void Margins)
- **熵增谷**: 业力不平衡导致的混乱区域，只有最勇敢的觉醒者敢于进入。
